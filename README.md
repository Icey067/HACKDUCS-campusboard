# CampusBoard

CampusBoard is a unified college notice and opportunity hub for Indian students.

**This is our official submission for the HACKDUCS 2026 Hackathon!** 🚀

## Overview

CampusBoard simplifies discovering notices, opportunities, and campus events by providing a centralized platform. 

## Structure

- `frontend/`: The React web application
- `backend/`: The Node.js Express server
